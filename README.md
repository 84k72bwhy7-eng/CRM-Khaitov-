## Portfolio Landing (Next.js)

This is a modern, responsive portfolio landing page built with **Next.js (App Router)**, **TypeScript**, and custom CSS.

### Running locally

1. Install dependencies:

   ```bash
   npm install
   # or
   pnpm install
   # or
   yarn
   ```

2. Start the dev server:

   ```bash
   npm run dev
   ```

3. Open `http://localhost:3000` in your browser.

### Customizing

- **Text/content**: Edit `app/page.tsx` (hero copy, about text, projects list, skills, contact details).
- **Colors/spacing/visual style**: Edit `app/globals.css` (CSS variables at the top and component styles below).
